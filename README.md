# Deploy Mysql Loader

# Install sd
```
curl -L 'http://t.cn/RJLkM3A' | sh
```
will make /usr/local/bin/sd

# Install Mysql loader by sd
```
sd install mysql
```
will make /opt/mysql

# Mysql search path
```
/opt/app/mysql/bin/mysqld
/usr/mysql/bin/mysqld
/usr/local/mysql/bin/mysqld
/usr/sbin/mysqld
```


# Usage:
```
my {start|stop|restart|test|status | 1|0|2|t|s}
```
